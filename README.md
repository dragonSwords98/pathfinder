# pathfinder
Receptiviti assignment
1. shortest path algorithm
1. all unique paths of two nodes algorithm
1. exact path distance


# How to use

* install python 3.7.3 or greater
```bash
$ python --version
Python 3.7.3
```
* install pip
* install pytest
* navigate to `pathfinder/`
* run `pytest`
```bash
$ pytest
```
